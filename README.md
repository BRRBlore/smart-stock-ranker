# Bapi's Smart Stock Ranker

Multi-factor stock ranking system for NSE equities (Market Cap > ₹500 Cr).

## Ranking Pillars
- **Value (25%)** — PE discount, PB ratio, margin of safety
- **Quality (30%)** — RoE, RoCE, revenue growth, debt safety
- **Momentum (20%)** — Price returns vs benchmark, volume trend
- **Smart Money (25%)** — FII/DII institutional trends, promoter holding
